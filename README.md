# 🎓 Career Katta Classes — Live Learning Platform

A full-stack **MERN Stack** web application that enables real-time 
live video sessions between hosts and participants — built for 
educators, students, and teams.

## 🚀 Live Demo
🌐 [career-katta-classes.vercel.app](https://career-katta-classes.vercel.app)

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT Authentication
- **Video:** ZegoCloud SDK
- **Deploy:** Vercel + Render

## ✨ Features
- 🔐 Secure Register & Login (JWT)
- 🎥 HD Live Video Sessions
- 💬 Real-time In-session Chat
- 🔗 Room ID based Session Sharing
- 👥 Host & Participant Management
- 📱 Fully Responsive UI
